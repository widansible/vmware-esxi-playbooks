Add User

1. Login to your vSphere Web Client ESXI Host
https://10.11.8.77
2. Create the User
Click on Host then Manage click on Security & Users then Users click on Add User , Add the name and password , and Click Add
3. Assign the Role
Now click on Host again then Click on Actions Go to Permissions , select the newly created User from the drop down Menu then select the required Role , Finally click on Add User.
You are done
